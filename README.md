# newDrinkAppFront

Aplicação front-end para gerenciamento de bebidas, com páginas para entrada, cadastro, painel, perfil e criação de novos itens.

## Funcionalidades
- Login e cadastro de usuário
- Navegação entre páginas principais da aplicação
- Visualização de bebidas no painel
- Criação, edição e exclusão de bebidas cadastradas
- Página de perfil do usuário

## Tecnologias
- HTML
- CSS
- JavaScript

## Estrutura do projeto
- `index.html`: página inicial
- `login.html`: tela de login
- `register.html`: tela de cadastro
- `dashboard.html`: painel principal
- `drink.html`: detalhes de bebida
- `create.html`: criação de bebida
- `profile.html`: perfil do usuário
- `css/styles.css`: estilos globais
- `js/`: scripts da aplicação

## Como rodar
1. Clone o repositório:
	```bash
	git clone https://github.com/cuaitz/newDrinkAppFront
	```
2. Abra a pasta do projeto no VS Code.
3. Inicie com o `Live Server` na página `index.html`.

> Observação: como o projeto usa arquivos JavaScript separados, é melhor executá-lo em um servidor local em vez de abrir o HTML direto no navegador.
